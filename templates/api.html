üöÄ Quick Start

This API provides endpoints to generate QR codes for UPI payments with advanced styling options.

# Basic QR Code Generation curl "http://localhost:8000/api/genqr?name=John%20Doe&id=john@paytm" # Advanced QR Code with Styling curl "http://localhost:8000/api/qrgen?upiid=john@paytm&name=John%20Doe&amount=100&size=500&fill_color=blue"

üìã Endpoints

GET /api/genqr

Generate a simple QR code for UPI payment (legacy endpoint for compatibility).

Parameters

Parameter Type Required Description
name string Required Payee name
id string Required UPI ID
amount number Optional Payment amount
size integer Optional QR code size (default: 300)
format string Optional Image format (default: png)

Example Request

GET /api/genqr?name=John%20Doe&id=john@paytm&amount=100&size=400

Example Response

{ "qr_code_data_url": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...", "upi_string": "upi://pay?pa=john@paytm&pn=John Doe&am=100", "size": 400, "format": "png" }

GET /api/qrgen

Generate an advanced QR code with styling options (returns PNG directly).

Parameters

Parameter Type Required Description
name string Required Payee name
upiid string Optional* UPI ID
ifsccode string Optional* IFSC code
accountnum string Optional* Account number
amount number Optional Payment amount
size integer Optional QR code size (default: 300)
fill_color string Optional Fill color (default: black)
back_color string Optional Background color (default: white)
logo_url string Optional Logo URL
style string Optional QR style: square, circle, rounded, diamond
format string Optional Image format: png, jpeg, svg

* Either upiid OR both ifsccode and accountnum must be provided

Example Request

GET /api/qrgen?upiid=john@paytm&name=John%20Doe&amount=100&size=500&fill_color=blue&style=rounded&logo_url=https://example.com/logo.png

POST /api/generate

Generate QR code with full customization options including file upload for logos.

Request Body

{ "upi_id": "john@paytm", "name": "John Doe", "amount": 100, "size": 500, "fill_color": "#0066cc", "back_color": "#ffffff", "logo_url": "https://example.com/logo.png", "logo_size_ratio": 0.3, "style": "rounded", "gradient": true }

Example Response

{ "qr_code_data_url": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...", "upi_string": "upi://pay?pa=john@paytm&pn=John Doe&am=100", "size": 500, "format": "png" }

GET /api/styles

Get available QR code styles and options.

Example Response

{ "styles": ["square", "circle", "rounded", "diamond"], "colors": ["black", "white", "red", "green", "blue", "yellow", "purple", "orange", "pink", "brown", "gray"], "formats": ["png", "jpeg", "svg"] }

üé® Styling Options

Available Styles

  • square - Standard square QR code
  • circle - Circular QR code elements
  • rounded - Rounded corners
  • diamond - Diamond-shaped QR code

Color Options

Colors can be specified as:

  • Hex colors: #ff0000, #00ff00
  • Named colors: red, blue, green
  • RGB values: rgb(255,0,0)

Logo Integration

Logos can be added via:

  • URL: logo_url parameter
  • File upload: Use POST endpoint with multipart form data

‚ö†Ô∏è Error Handling

The API returns appropriate HTTP status codes and error messages:

{ "error": "Invalid UPI ID format", "message": "Please provide a valid UPI ID" }

Common error codes:

  • 400 - Bad Request (invalid parameters)
  • 429 - Too Many Requests (rate limited)
  • 500 - Internal Server Error

üîí Rate Limiting

API requests are rate limited to prevent abuse:

  • 100 requests per minute per IP address
  • Rate limit headers are included in responses
  • Exceeded limits return 429 status code
